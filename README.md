# Hiring-Pipeline-Automation
An automation tool for HRs which has all hiring processes in a single dashboard, managing hiring for different roles, from shortlisting resumes to scheduling interviews.

Setup for Google Form integration:
1. Create your Google Form with these fields:
   - Full name (Short answer)
   - Email (Short answer)  
   - Resume (File upload)
2. Open the linked Google Sheet → Share with:
   hiring-bot@your-project.iam.gserviceaccount.com
3. Go to Google Drive → Find the folder "Form responses [your form name]"
   → Share that folder with the same service account email
4. Paste the Sheet ID into the tool
